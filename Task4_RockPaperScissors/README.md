# Task 4 - Rock Paper Scissors Game

This project is created as part of the Python Internship at CodSoft.

## About the Project
This is a simple Rock Paper Scissors game built using Python.
The user plays against the computer by choosing rock, paper, or scissors.

## How It Works
- The user enters a choice: rock, paper, or scissors.
- The computer randomly selects an option.
- The winner is decided based on standard game rules.
- Invalid inputs are handled properly.

## Technologies Used
- Python
- Random module
